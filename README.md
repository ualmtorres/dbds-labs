# dbds-labs

Repositorio de laboratorios para el seminario "Bases de Datos para Ciencia de Datos".

## Objetivo
Reunir ejercicios prácticos sobre modelado, consultas, optimización y manejo de datos para apoyar el aprendizaje.

## Contenido
- Ejemplos de SQL.
- Scripts de preparación de datos.
- Notebooks exploratorios.
- Casos de análisis de rendimiento.

## Requisitos
- Python 3.x (si se usan notebooks).
- Motor de base de datos (PostgreSQL recomendado).
- Git para control de versiones.

## Cómo empezar
1. Clonar el repositorio.
2. Revisar la carpeta de ejercicios.
3. Seguir las instrucciones de cada laboratorio.

## Estructura (propuesta)
```
/sql
/notebooks
/datasets
/scripts
```
