# dbds-labs

Repositorio de laboratorios para el seminario "Bases de Datos para Ciencia de Datos".

## Objetivo
Reunir ejercicios prácticos sobre consultas y manejo de datos para apoyar el aprendizaje.

## Contenido
- Ejemplos de SQL.
- Scripts de preparación de datos.
- Notebooks exploratorios.

## Requisitos
- Para entorno online:
  - Google Colab para notebooks.
  - Servidor de base de datos accesible (p.e. MySQL en la nube con un proveedor como Aiven)
  - Librerías Python: `sqlalchemy`, `pymysql` y `jupysql`.
- Para entorno local:
    - Docker y Docker Compose instalados para levantar un entorno con Jupyter y bases de datos.
- Para entorno local sin Docker:
    - JupyterLab instalado.
    - Servidor PostgreSQL o MySQL local o remoto accesible.
    - Librerías Python: `sqlalchemy` y `pymysql`.

## Cómo empezar
1. Clonar el repositorio.
2. Revisar la carpeta de ejercicios.
3. Seguir las instrucciones de cada laboratorio.
